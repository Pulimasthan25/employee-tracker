🚀 EmployeeTracker

A modern employee tracking application built with Angular CLI. This project helps manage and monitor employee activities with a scalable and maintainable frontend architecture.


---

📌 Features

📊 Track employee activity and work logs

⚡ Fast and reactive UI with Angular

🧩 Modular and scalable structure

🧪 Unit testing support using Vitest

🔄 Hot reload during development



---

🛠️ Tech Stack

Frontend: Angular (v21+)

Build Tool: Angular CLI

Testing: Vitest

Language: TypeScript



---

🚀 Getting Started

Prerequisites

Make sure you have installed:

Node.js (v18 or higher recommended)

npm (comes with Node)



---

📥 Installation

Clone the repository:

git clone <your-repo-url>
cd EmployeeTracker

Install dependencies:

npm install


---

▶️ Run the Application

Start the development server:

ng serve

Open your browser and go to:

http://localhost:4200/

The app will automatically reload when you make changes.


---

⚙️ Project Structure (High-Level)

src/
│── app/
│   ├── components/     # Reusable UI components
│   ├── services/       # Business logic & API calls
│   ├── models/         # Data models/interfaces
│   ├── pages/          # Feature pages
│   └── app.module.ts   # Root module
│
└── assets/             # Static files


---

🧩 Code Generation

Generate components, services, etc.:

ng generate component component-name
ng generate service service-name

See all options:

ng generate --help


---

📦 Build for Production

ng build

Build output will be available in:

dist/


---

🧪 Running Tests

Run unit tests:

ng test


---

🔍 End-to-End Testing

ng e2e

> Note: You need to configure an e2e testing framework (like Cypress or Playwright).
